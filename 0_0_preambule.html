1 Data Science avec R

R est un excellent pour pratiquer la data science.

1.1 Installation de R

  • Téléchargement du logicile R sur le site r-project
  • Installation de RStudio

Pour RStudio, si vous télécharger le fichier zippé, vous n’avez pas besoin de lancer le programme d’installation de Windows.

De nombreux tutoriels pour débuter R sont disponibles sur internet.

1.3 Bonnes pratiques

Quelques conseils pour débuter ou tout petits projets dans R:

  • Création d’un fichier rmarkdown
  • Commentaire pour les codes
  • Sauvegarde des fichiers de données
  • Compilation du fichier rmarkdown pour la reproductibilité
